Detailed Description

Abstract class representing a vertex array in the rendering system.

The VertexArray class manages the collection of vertex buffers and an optional index buffer. It provides the interface for binding, unbinding, and configuring vertex attributes in the rendering pipeline.

Responsibilities:

  • Manage vertex buffers and index buffers.
  • Bind/unbind the vertex array for rendering.
  • Provide access to underlying buffers.

Derived classes (e.g., OpenGlVertexArray) implement platform-specific behavior for managing vertex arrays.
