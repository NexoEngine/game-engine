Detailed Description

Base interface for all component array types.

Provides the common interface that all concrete component arrays must implement, allowing type-erased storage and manipulation of components.

Note
This class is not thread-safe. Access to a ComponentArray should be synchronized externally when used in multi-threaded contexts.
