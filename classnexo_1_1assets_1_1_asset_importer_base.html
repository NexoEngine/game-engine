virtual bool nexo::assets::AssetImporterBase::canRead ( const ImporterInputVariant &  inputVariant)
pure virtual

Checks if the importer can read the file at the given path.

Implementations should open the file and check if related importer is compatible.

Parameters
[in]inputVariantThe input variant to check. Can be a file path or memory buffer.
Returns
True if the importer can read the file, false otherwise.

Implemented in nexo::assets::ModelImporter, and nexo::assets::TextureImporter.

◆ import()

void nexo::assets::AssetImporterBase::import ( AssetImporterContext &  ctx)
inlinenoexcept

Imports an asset from a file.

This method is not intended to be overridden. Implement importImpl() to do the import. This method is a wrapper of importImpl() that for example catches exceptions thrown by importImpl().

Parameters
[in,out]ctxThe context for the import.
