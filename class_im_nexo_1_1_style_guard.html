Detailed Description

Guard class for managing ImGui style colors.

Automatically pushes ImGui style colors on construction and pops them on destruction, ensuring the style state is properly restored even when exceptions occur. Supports chaining multiple color changes with the push() method.

Delete copy constructor since we can't precisely duplicate the pushed styles.

We only track which color indices were pushed, but not their actual values. Without storing the color values, we can't properly recreate the pushed styles.
