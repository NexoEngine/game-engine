Detailed Description

Abstract class for input handling.

Provides an interface for querying keyboard and mouse input states. Implementations (e.g., for OpenGL using GLFW) must provide concrete behavior for checking key and mouse states as well as retrieving the mouse position.

Usage:

  • Call Input::init(window) to initialize the input instance.
  • Use the static inline functions (e.g., isKeyPressed) for convenience.
