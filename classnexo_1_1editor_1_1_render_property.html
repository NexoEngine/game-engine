void nexo::editor::RenderProperty::show ( ecs::Entity  entity)
overridevirtual

Displays and manages the render properties UI for the specified entity.

This function retrieves the render component of the given entity and renders a user interface for its render properties using ImGui. It configures the material inspector for 3D entities and sets up a placeholder for 2D entities. When the material section is active, a scene preview is generated, allowing the user to view the material in a framebuffer. The UI also offers controls to toggle entity visibility, select a material type from a dropdown, and open popups to create or modify materials.

Parameters
entityThe entity whose render properties are to be displayed.

Implements nexo::editor::IEntityProperty.

Displays a popup modal for creating a new material.

Renders a two-column popup using ImGui where the left panel accepts material inputs (name and properties) via a material inspector, and the right panel shows a live preview of the material rendered in a framebuffer. If a preview scene hasnâ€™t been generated yet, it is created for the specified dimensions. When the user confirms by clicking "OK", the material is linked to the entity's render component and the preview scene is cleaned up; clicking "Cancel" simply deletes the preview scene.
