Detailed Description

Manages multiple scenes within the engine.

The SceneManager is responsible for creating, storing, and deleting scenes. It also provides access to individual scenes by their unique ID.

Responsibilities:

  • Creating both runtime and editor-only scenes.
  • Deleting scenes.
  • Providing access to scenes by ID.

Note: A valid ECS Coordinator must be set via setCoordinator() before creating scenes.
