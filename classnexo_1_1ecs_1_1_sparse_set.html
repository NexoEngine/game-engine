Detailed Description

A sparse set implementation for efficient entity storage and lookup.

This class provides O(1) insertion, removal, and lookup operations for entities. It uses a sparse-dense pattern where entities are stored contiguously in a dense array, while maintaining a sparse lookup map to quickly find entity positions.
