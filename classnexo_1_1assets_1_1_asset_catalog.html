◆ registerAsset()

GenericAssetRef nexo::assets::AssetCatalog::registerAsset ( const AssetLocation &  location,
std::unique_ptr< IAsset >  asset 
)

Registers an asset in the catalog.

This method verifies that the provided asset pointer is valid, then creates a shared pointer to the asset. It updates the asset's metadata by setting its location and assigning a new unique identifier if one is not already set. The asset is stored in the catalog and a reference to the asset is returned.

Warning
Once registered, the memory for the asset is managed by the catalog. Do not delete the asset.
Parameters
locationThe asset's location metadata.
assetPointer to the asset to be registered.
Returns
GenericAssetRef A reference to the registered asset, or a null reference if the asset pointer was null.
