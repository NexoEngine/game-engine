Detailed Description

template<typename... EventTypes>
class nexo::event::Listens< EventTypes >

Variadic class for listeners that can handle multiple event types.

The Listens class inherits from BaseListener and from Listener for each EventType provided. This allows a single object to listen to and handle multiple types of events.
