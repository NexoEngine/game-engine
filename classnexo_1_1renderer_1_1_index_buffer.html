Detailed Description

Abstract class representing an index buffer in the graphics pipeline.

An index buffer stores indices into a vertex buffer, allowing for efficient reuse of vertex data. This class provides an abstract interface for creating, binding, and managing index buffers, enabling compatibility with multiple graphics APIs.

Binding the index buffer ensures that subsequent rendering commands will use the indices stored in this buffer to draw primitives.
