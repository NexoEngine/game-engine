void nexo::editor::WindowRegistry::registerWindow ( std::shared_ptr< T >  window)
inline

Adds a document window instance to the registry.

This function registers a document window of type T by storing its shared pointer in an internal map keyed by the window's type identifier. Before adding the new window, it checks whether a window with the same name is already registered under the same type. If such a window exists, a WindowAlreadyRegistered exception is thrown to prevent duplicate entries.

Template Parameters
TThe concrete window type derived from IDocumentWindow.
Parameters
windowA shared pointer to the window instance to be registered.

Retrieves a registered window of the specified type and name.

Retrieves a range view of document windows cast to a specified derived type.

This template function checks whether a window of type T with the given name has been registered. If the window is found, it returns the window as a weak pointer of type T. Otherwise, it returns an empty weak pointer.

Template Parameters
TThe type of the window, which must derive from IDocumentWindow.
Parameters
windowNameThe unique name of the window to look for.
Returns
std::weak_ptr<T> A weak pointer to the registered window if found; otherwise, an empty weak pointer.

This function returns a std::ranges::transform_view over the vector of document windows stored in m_windows for the type T. The transformation is performed using a non-capturing helper function, castWindow<T>, which casts each std::shared_ptr<IDocumentWindow> to a std::shared_ptr<T>.

If no windows of the requested type T are found in m_windows, an empty range is returned.

Template Parameters
TThe derived type of IDocumentWindow to retrieve.
Returns
A transform_view range over the vector of document windows cast to std::shared_ptr<T>.
