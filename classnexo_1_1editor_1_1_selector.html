Detailed Description

Singleton class managing entity selection state in the editor.

The Selector class tracks the currently selected entities, their types, and provides methods to manipulate the selection state. It also maintains entity UUID to UI handle mappings for consistent labeling in the interface.

Gets the primary selected entity.
