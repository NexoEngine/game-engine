Constructs a new vertex buffer and initializes it with vertex data.

This constructor creates a new OpenGL vertex buffer, allocates GPU memory, and uploads the provided vertex data to the buffer.

Parameters
verticesPointer to the array of vertex data to initialize the buffer with.
sizeThe size (in bytes) of the vertex data to upload.

OpenGL Calls:

  • glGenBuffers: Generates a new buffer object.
  • glBindBuffer: Binds the buffer as the current vertex buffer (GL_ARRAY_BUFFER).
  • glBufferData: Allocates GPU memory and uploads the vertex data.

nexo::renderer::NxOpenGlVertexBuffer::NxOpenGlVertexBuffer ( unsigned int  size)
explicit

Constructs an empty vertex buffer with a specified size.

This constructor creates a new OpenGL vertex buffer and allocates GPU memory without uploading any data. The buffer is intended for dynamic updates.

Parameters
sizeThe size (in bytes) of the buffer to allocate.

OpenGL Calls:

  • glGenBuffers: Generates a new buffer object.
  • glBindBuffer: Binds the buffer as the current vertex buffer (GL_ARRAY_BUFFER).
  • glBufferData: Allocates GPU memory with a nullptr for data.

Usage:

  • Call setData later to populate the buffer with vertex data dynamically.

This method replaces the content of the vertex buffer with new data. The data must fit within the allocated buffer size.
