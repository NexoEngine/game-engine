Detailed Description

Represents an individual element in a buffer layout.

Each buffer element specifies its name, type, size, and offset within a buffer. This struct is used to define the layout of data in a vertex buffer.

Members:

  • Parameters
    nameThe name of the buffer element (e.g., "position" or "color").
  • Parameters
