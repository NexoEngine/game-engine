Detailed Description

OpenGL-specific implementation of the NxWindow class.

The NxOpenGlWindow class manages the creation and behavior of a window in an OpenGL context. It integrates with GLFW for window management and event handling.

Responsibilities:

  • Create and manage an OpenGL-compatible window.
  • Provide event handling for window, keyboard, and mouse events.
  • Manage OpenGL context initialization and VSync settings.
