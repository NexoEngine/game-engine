Detailed Description

AssetLocation is a wrapper of std::string to represent the location of an asset.

It is used to apply our own rules on naming

void nexo::assets::AssetLocation::setLocation ( const std::string &  fullLocation)
inline

Parses and sets the asset's location from a full location string.

Extracts the asset name, path, and optional pack name from the provided string and updates the corresponding internal members. The expected format is "packName::name@path" or "name@path" if no pack name is included. If the extracted asset name is invalid, an InvalidAssetLocation exception is thrown.

Parameters
fullLocationFull asset location string.
Exceptions
InvalidAssetLocationif the asset name in the provided string is invalid.

Compares the current asset location with a full location string.

Parses the provided full location string into its asset name, path, and pack name components and compares them with the object's corresponding values. Returns true if all components match, indicating that the asset location is equivalent to the provided string.
