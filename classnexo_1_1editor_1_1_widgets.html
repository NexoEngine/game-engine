Detailed Description

A collection of custom ImGui widget drawing functions.

Provides utility functions for drawing color editors, texture buttons, and a material inspector, which can be used to simplify UI code for rendering material properties.

Draws a color editor with a button and an optional inline color picker.

Displays a custom color button (with a cog icon for picker settings) and, if enabled, an inline color picker. The function returns true if the color was modified.

Parameters
labelA unique label identifier for the widget.
selectedEntityColorPointer to the glm::vec4 representing the current color.
colorPickerModePointer to the ImGuiColorEditFlags for the picker mode.
showPickerPointer to a boolean that determines if the inline color picker is visible.
colorButtonFlagsOptional flags for the color button (default is none).
Returns
true if the color was modified; false otherwise.

Draws a texture button that displays a texture preview.

When clicked, opens a file dialog to select a new texture. If a new texture is loaded, the passed texture pointer is updated and the function returns true.

Parameters
