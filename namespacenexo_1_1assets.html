◆ from_json()

void nexo::assets::from_json ( const nlohmann::json &  j,
AssetType &  type 
)
inline

Converts a JSON value to its corresponding AssetType.

This function iterates over the predefined asset type names and, if the JSON value matches one, assigns the corresponding AssetType enum value to the output parameter. If no match is found, the AssetType is set to UNKNOWN.

Parameters
jJSON object containing the asset type as a string.
typeOutput parameter to store the resulting AssetType.
