Detailed Description

Implementation of the Shader interface using OpenGL.

The NxOpenGlShader class provides OpenGL-specific functionality for creating and managing shader programs. It supports setting uniform variables and compiling shaders from source code or files.

Responsibilities:

  • Compile and link shader programs using OpenGL.
  • Set uniform variables for rendering operations.
  • Manage the lifecycle of OpenGL shader programs.

nexo::renderer::NxOpenGlShader::NxOpenGlShader ( const std::string &  path)
explicit

Constructs a shader program from a source file.

Reads and compiles a shader program from the specified file path. The file should contain #type directives to separate shader stages.

Parameters
pathThe file path to the shader source code.

Throws:

  • NxFileNotFoundException if the file cannot be found.
  • NxShaderCreationFailed if shader compilation fails.
