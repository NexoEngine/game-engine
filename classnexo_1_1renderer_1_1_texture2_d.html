Creates a blank 2D texture with the specified dimensions.

Allocates a texture resource with the given width and height. The texture will have no initial data and can be updated later with setData.

Parameters
widthThe width of the texture in pixels.
heightThe height of the texture in pixels.
Returns
A shared pointer to the created Texture2D instance.

Example:

auto blankTexture = Texture2D::create(512, 512);

std::shared_ptr< Texture2D > nexo::renderer::Texture2D::create ( const std::string &  path)
static

Creates a 2D texture from an image file.

Loads the texture data from the specified file path. The file must contain image data in a supported format (e.g., PNG, JPG). The texture will be ready for rendering after creation.

Parameters
pathThe file path to the texture image.
Returns
A shared pointer to the created Texture2D instance.

Example:

auto texture = Texture2D::create("assets/textures/brick_wall.png");
