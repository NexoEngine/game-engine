Detailed Description

ManagedApi struct to hold function pointers to managed API functions.

This struct is used to hold function pointers to managed API functions that can be called from native code.

Warning
At runtime, all functions MUST be initialized to a none nullptr value. Otherwise, an exception will be thrown by HostHandler::checkManagedApi().
